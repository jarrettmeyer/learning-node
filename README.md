# Just me, learning NodeJS

## Sample App #1

To run Sample App #1...

    $ cd ./sample_app_1
    $ node ./server.js

To stop the sample app, use `CTRL-c`.

### Unit Tests

The unit tests for Sample App #1 are written in [Mocha](http://visionmedia.github.io/mocha/). To run the unit tests for Sample App #1...

    $ npm install -g mocha
    $ cd ./sample_app_1
    $ ./run_tests.sh
